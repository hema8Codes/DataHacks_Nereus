# Species impact during the Tasman Sea 2017-18 heatwave

## Summary
Tasman events drove some of the best-documented poleward range shifts in the global literature.

## Key observations
Mass mortality of abalone, commercial lobster recruitment dropped sharply, kelp forest contraction accelerated. iNaturalist captured ~780 distinct species in observations.

## Coverage
Tasman Sea is a warming hotspot — one of the top-ten fastest-warming marine regions globally.

## Biological lesson
Subtropical species are colonizing Tasmanian waters year-round; temperate specialists are running out of southward refuge.

## Data source
iNaturalist research-grade observations, accessed via iNaturalist API v1, filtered to marine iconic taxa (Actinopterygii, Mollusca, Mammalia, Aves) within the canonical event bounding box. Each event window is paired with a matching baseline window from a climatologically similar non-event year for anomaly computation.
